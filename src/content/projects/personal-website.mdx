This portfolio completely reimagines a standard website as a fully interactive Unix-style terminal emulator. Built using an Astro static shell that hydrates a pure React terminal engine, the site delivers a unique, nostalgic experience without sacrificing modern maintainability or SEO.

## Features

- **Custom Terminal Engine**: The core `Terminal.tsx` component handles keyboard event loops, ArrowUp/ArrowDown command history, Tab-completion, and dynamic output routing.
- **16 Internal Commands**: Visitors can run commands like `resume`, `projects`, `skills`, and `socials` to view cleanly formatted output.
- **Pure CSS Styling**: Ditching external UI libraries, the site relies entirely on vanilla CSS and custom properties to achieve a CRT scanline effect, text glow plugins, and dynamic color switching. 
- **4 Live Themes**: Users can run `themes set amber` or `dracula` to instantly swap the root color variables, persisting their choices via `localStorage`.

## Architecture

To ensure the content remains easy to update, the terminal commands read directly from Astro's data layer:
- **MDX Collections**: `blog` and `projects` are written in MDX. Astro statically parses these at build time and passes the frontmatter into the React terminal component as props.
- **JSON Timelines**: The `resume` command dynamically parses standalone JSON files representing each career stop, filtering them smoothly when users run subcommands like `resume experience`.

The result is a highly engaging, interactive landing page that still builds to a lightning-fast static directory hosted for free on GitHub Pages.
